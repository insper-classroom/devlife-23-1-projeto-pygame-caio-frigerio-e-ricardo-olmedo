Dragons rage

Dragons rage foi desenvolvido como projeto do curso de Ciência da Computação do Insper,
tendo como inspiração o clássico jogo da cobrinha do Nokia, mas com algumas modificações.
O objetivo é simples: comer os cidadãos que valem 10 pontos cada um e recuperam uma vida,
derrotar os guardas que causam 2 pontos de dano. O jogador começa com 1 pontos de vida e os
guardas, quando derrotados, dobram os pontos atuais do jogador.
Desafie-se e veja quantos pontos consegue acumular!

Como jogar

Como este jogo foi baseado no classico jogo da cobrinha, para jogar e simples voce deve utilizar as setas do teclado.
A seta para cima faz com que o dragão suba, a seta para baixo faz com que o dragão desça, a seta para a esquerda
faz com que o dragão vá para a esquerda e a seta para a direita faz com que o dragão vá para a direita.
O objetivo do jogo é comer o maior numero cidadoes e matar os guardas.
